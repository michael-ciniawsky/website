## Website

## Setup
Install [Node](http://nodejs.org) && [npm](http://npmjs.org)

## Install

```bash
$ (sudo) npm i
```

## Build

```bash
$ npm run build
```

## Serve

```bash
$ npm run serve // NODE_ENV=Development
```

## Bundle

```bash
$ npm run bundle  // NODE_ENV=Production
```

## Server

```bash
$ npm start
```
